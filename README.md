## Run Locally

Clone the project

```bash
  git clone https://github.com/matruh/MRuhiyatUKK.git
```

Go to the project directory

```bash
  cd perpustakaan
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
